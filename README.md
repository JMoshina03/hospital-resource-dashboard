# Hospital Resource Utilization & Patient Outcomes Dashboard

## Problem Statement
Hospitals face challenges in monitoring admissions, ICU demand, bed occupancy, doctor utilization, and revenue across departments and branches.

## Solution
This project provides an interactive dashboard using MySQL and Power BI to analyze hospital resource utilization and patient outcomes.

## Tech Stack
- MySQL
- Power BI
- GitHub

## Key KPIs
- Total Admissions
- Average Length of Stay (ALOS)
- ICU Requirement %
- Readmission Rate
- Total Revenue
- Bed Occupancy Rate

## Dataset
The dataset is available in the `/dataset` folder.
- patients.csv
- admissions.csv
- billing.csv
- doctors.csv
- beds.csv

## SQL
All database creation, KPI queries, and dashboard views are available in `/sql/hospital_db.sql`.

## Dashboard
Power BI dashboard file is included in the repository.

## Outcome
This dashboard helps hospital management optimize staffing, beds, ICU resources, and financial planning.
